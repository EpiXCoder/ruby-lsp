### `undef`

[Read this documentation in the browser](https://github.com/Shopify/ruby-lsp/blob/main/static_docs/descriptions/undef.md)

The `undef` keyword is used to undefine a method, making it unavailable for use.

```ruby
class MyClass
  def my_method
    puts "Hello"
  end

  undef my_method
end
```
