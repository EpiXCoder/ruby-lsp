### `class`

[Read this documentation in the browser](https://github.com/Shopify/ruby-lsp/blob/main/static_docs/descriptions/class.md)

The `class` keyword is used to define a new class.

```ruby
class MyClass
  def initialize(name)
    @name = name
  end
end
```
